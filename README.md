# hackerton1-weather

# 제1회 iOS 1인 해커톤 - 주제 : 날씨 앱

### 프로젝트 목적
OpenAPI를 사용해서 날씨 정보를 받아오는 ios 앱을 만든다

### 외부 package
- realm
- firestore(firebase)
- swiftUIPager
- lottie
- alamofire

### 기술 및 패턴
- MVVM
- coordinator pattern
- SwiftUI
- Swift
- Combine

### 페이지 정보
1. 날씨 정보 페이지
2. 설정 페이지
3. 지역 선택 페이지
4. 날씨별 게시판 페이지
